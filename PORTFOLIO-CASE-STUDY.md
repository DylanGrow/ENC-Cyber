# ENC252 CyberSec Hub – Portfolio Case Study
**Dylan Grow | New Bern, NC**

## Overview
Static-first community platform for 252 area cybersecurity pros, students, veterans. Built for GitHub Pages.

## Features
- Forum with topics, threads, moderation
- Job Board (12 local listings, filters, JSON-LD)
- Meetups (calendar, RSVP, .ics)
- Projects (collaboration board)
- Newsletter, profiles, dark mode

## Tech
HTML/CSS/JS only, WCAG AA, SEO-ready, localStorage demo, designed to swap to Supabase API.

## Why ENC252
Supports ECU, Craven CC, FTCC, Camp Lejeune pipelines and Carolina Cyber Network 3,000-student goal.

## Deploy
1. Upload files
2. Enable GitHub Pages
3. Optional custom domain via CNAME
